# Twitter 简报质量评估标准 v1.0

## 优质推文评估维度

### 1. 信源权重 (40%)
- **KOL 级别**: 行业知名创始人/投资人 (如郭宇、Vitalik)
- **数据可信度**: 有实际数据支撑，非主观猜测
- **历史准确率**: 该账号过往预测/分析的准确率

### 2. 信息时效性 (25%)
- **突发新闻**: 1小时内首发
- **独家信息**: 未被广泛传播的一手信息
- **时效窗口**: 交易/投资机会的时效性

### 3. 内容价值 (25%)
- **可执行性**: 能否转化为具体行动
- **认知增量**: 是否提供新的视角/信息
- **影响范围**: 对行业/市场的潜在影响

### 4. 社区互动 (10%)
- **转发/点赞比**: 高互动 = 高共识
- **评论区质量**: 讨论深度和专业度

## 分类标签
- 🚨 **紧急**: 需立即行动 (安全预警、重大机会)
- 💡 **洞见**: 认知升级
- 🛠️ **工具**: 可立即使用的产品/服务
- 📊 **数据**: 市场/链上数据
- 🎭 **叙事**: 热点/情绪分析

## 简报格式标准
1. 评分 (1-10)
2. 分类标签
3. 核心要点 (3点以内)
4. 原文链接
5. 行动建议
