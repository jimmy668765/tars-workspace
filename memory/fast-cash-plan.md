### 2.2 市值<1w的筛选标准

**红名单(立即排除)**:
- [ ] 合约有mint功能且未放弃
- [ ] 流动性未锁定或锁定<30天
- [ ] 开发者钱包持有>10%
- [ ] 同名/相似名已存在高市值币
- [ ] 社交媒体账号<7天
- [ ] 网站是模板站且未完成

**黄名单(谨慎观察)**:
- [ ] 持有者<50人
- [ ] 24h交易量<1K
- [ ] 只有一个CEX钱包在买
- [ ] 没有Telegram/Discord群

**绿名单(重点监控)**:
- [ ] 社区活跃(TG群>100人且真实)
- [ ] 叙事新颖(非简单仿盘)
- [ ] 有KOL喊单但市值还低
- [ ] 合约干净, 开发者锁仓
- [ ] 有路线图/白皮书(即使是简单的)

### 2.3 叙事评分量化指标

**叙事评分模型 (满分100)**:

```
社区热度 (30分):
├── Twitter讨论量: 0-10分 (最近24h提及数)
├── Telegram活跃度: 0-10分 (消息频率/人数比)
└── KOL关注度: 0-10分 (知名交易员关注数)

叙事新颖度 (25分):
├── 概念独特性: 0-10分
├── 与当前热点关联: 0-10分
└── 可传播性: 0-5分 (有无meme素材)

代币经济学 (25分):
├── 公平发射: 0-10分
├── 持有者分布: 0-10分
└── 流动性充足度: 0-5分

技术基础 (20分):
├── 合约安全: 0-10分
├── 有无实用功能: 0-5分
└── 开发团队透明度: 0-5分
```

**操作SOP**:
1. 评分>70: 小仓位试水 (10-50U)
2. 评分>80: 标准仓位 (100-200U)
3. 评分>90 + 有CX迹象: 重仓(500U+)
4. 24h内不涨反跌>30%: 止损
5. 翻倍后出本金, 利润博更高
