## 🔥 核心认知（从高手那里学到的）

### 1. Timing > 一切
- Meme币不是价值投资，是**注意力套利**
- 关键不是币好不好，是**什么时候有人关注**
- 高手只在两种时候出手：
  - **热点前夜**（信息差窗口）
  - **启动瞬间**（成交量异动确认）

### 2. 信息源分层
**Tier 1: 原始热点源**
- 名人推特（Elon、CZ等）
- 重大新闻（DeepSeek发布、政策变化）
- 链上大额异动（聪明钱集体买入）

**Tier 2: 情绪放大器**
- KOL讨论热度
- 社群活跃度（TG群、Discord人数增长）
- Google搜索趋势

**Tier 3: 链上数据验证**
- 新钱包数量激增
- 持仓集中度变化
- 交易所流入/流出

### 3. 入场纪律（血泪教训）
| 条件 | 要求 | 违反后果 |
|------|------|----------|
| 市值 | $10K-$500K | 太高没空间，太低没流动性 |
| 成交量 | 24h突增>300% | 确认资金进入 |
| 叙事 | 匹配当前热点 | 没叙事的币死得快 |
| 仓位 | 单笔<5%总资金 | 归零也能活下去 |
| 止损 | 硬止损-50% | 被套就永久丧失机会成本 |

### 4. 出场策略（比入场更重要）
**分批出货：**
- 2x → 出本金（利润继续赌）
- 5x → 出50%利润
- 10x → 清仓或留10%底仓

**绝不做的：**
- 回本不卖（这是大忌）
- 等"更高"（顶部只有事后知道）
- 跌了补仓（Meme币归零是常态）

## ⚠️ 高手的禁忌

1. **不逆势扛单** → 错了就认，马上走
2. **不FOMO追高** → 已经涨3倍的不碰
3. **不All in** → 永远留子弹等下一个
4. **不情绪化** → 亏了不影响下一次判断
5. **不持币过夜** → 除非已经有大利润垫
