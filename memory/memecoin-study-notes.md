# TARS 学习笔记 - Memecoin 投机研究报告

## 核心认知
> **速度 > 信息**。T+1 到 T+6 小时是黄金窗口，超过 24 小时入场大概率亏钱。

## 关键数据
| 入场时间窗 | 平均收益率 | 胜率 | 策略 |
|-----------|-----------|------|------|
| 0-1小时 | 3,200% | 28% | 仅适合量化/自动化 |
| 1-6小时 | 890% | 22% | **最优风险收益比** |
| 6-24小时 | 245% | 15% | 需严格筛选 |
| 24h+ | -23% | 3% | **避免** |

## 市场操纵警示
- 82.8% 的高收益 memecoin 存在市场操纵迹象
- 62.9% 的跑路项目之前有洗盘交易
- 平均每个受害者损失约 $547

## 可执行的打法（与 TARS 能力结合）

### 第一层：自动化监控（TARS 可搭建）
- 链上流动性创建警报（<30秒延迟）→ `cron` + `browser`
- Smart Money 买入推送（<60秒延迟）→ 需 API 或浏览器监控
- 社媒提及量异常检测（<5分钟延迟）→ `web_search` / 浏览器

### 第二层：快速验证（TARS 可辅助）
- 合约安全检查（TokenSniffer）→ 浏览器自动化
- 持仓分布分析 → 截图/数据抓取
- 叙事质量评估 → AI 分析

### 第三层：执行与风控（TARS 可提醒）
- 一键买入（需交易机器人如 Banana Gun）
- 自动止盈/止损提醒 → `cron` 定时检查
- 持仓监控与预警 → 自动化脚本
